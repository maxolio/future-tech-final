# Future Tech AI News

Адаптивный многостраничный сайт о технологиях будущего.

## Технологии
- **HTML5**: Семантика и доступность (ARIA).
- **SCSS**: БЭМ (BEM), Fluid Design (`clamp`), модульность.
- **JS**: ООП подход, работа с состоянием (`Proxy`).
- **Сборка**: Dart Sass.

## Запуск проекта

1. **Клонировать репозиторий:**
   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/future-tech.git
   ```

2. **Установить зависимости:**
   ```bash
   npm install
   ```

3. **Запустить компиляцию стилей:**
   ```bash
   npm run sass-watch
   ```

4. **Открыть в браузере:**
   `index.html` (рекомендуется через Live Server).

## Особенности
- Плавные аккордеоны на CSS Grid.
- Независимые модули табов и селектов.
- Полный адаптив без громоздких медиа-запросов.
```

**Кратко и по делу.** Такой формат лучше всего подходит для быстрой оценки проекта. Не забудь только вставить свою ссылку на гитхаб в первом пункте.
